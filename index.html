关于作者

Bootstarlight 新框架是由 Franki Yu 抽取整合 Bootstrap 常用组件，在此基础上设计了多款样式，收集了目前常用的JavaScript组件并对跟Bootstrap的兼容性做了修改。

如果你发现本框架中有任何错误，敬请指正。通过下方的反馈按钮或访问 Open an issue on GitHub 为本框架添加或贡献内容。
